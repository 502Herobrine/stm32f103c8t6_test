基于 STM32F103C8T6（中容量增强型，LQFP48 封装）的规格书，为您整理了一份引脚使用指南。

### 1. 核心系统引脚
这些引脚是单片机运行的基础，必须正确连接：

* **电源引脚**:
    * **$V_{DD}$ / $V_{SS}$**: 数字电源及地。工作电压范围为 **2.0V 至 3.6V** [cite: 2, 359]。
    * **$V_{DDA}$ / $V_{SSA}$**: 模拟电源及地。为 ADC 和复位模块供电，通常应与 $V_{DD}$ 保持一致 [cite: 384, 196]。
    * **$V_{BAT}$**: 电池供电引脚（1.8V 至 3.6V）。当主电源断电时，为实时时钟（RTC）和后备寄存器供电 [cite: 2, 272]。
* **复位与时钟**:
    * **NRST**: 低电平有效复位引脚 [cite: 382]。
    * **OSC_IN / OSC_OUT**: 接 4MHz 至 16MHz 的外部高速晶振（HSE） [cite: 2, 267]。
    * **OSC32_IN / OSC32_OUT**: 接 32.768kHz 外部低速晶振（LSE），供 RTC 使用 [cite: 2, 270]。
* **启动配置**:
    * **BOOT0**: 启动模式选择位。配合 BOOT1（通常映射在 PB2）决定是从 Flash、系统存储器（串口下载模式）还是内置 SRAM 启动 [cite: 41]。

### 2. 调试与下载接口
STM32 支持两种调试接口，推荐使用 **SWD** 模式以节省引脚 [cite: 284]：

* **SWD 模式 (推荐)**:
    * **PA13**: SWDIO (数据) [cite: 382, 39]
    * **PA14**: SWCLK (时钟) [cite: 382, 39]
* **JTAG 模式**:
    * **PA15 (JTDI)**, **PB3 (JTDO)**, **PB4 (NJTRST)** 以及上述 PA13/PA14 [cite: 382]。

### 3. 常用外设引脚映射 (LQFP48)
STM32F103C8T6 的 I/O 具有高度的可复用性。以下是常用功能的默认引脚分配 [cite: 382, 383, 384]：

| 功能模块 | 默认引脚 (LQFP48) | 说明 |
| :--- | :--- | :--- |
| **USART1 (串口1)** | **TX: PA9**, **RX: PA10** | 常用于程序下载和日志打印 |
| **USART2 (串口2)** | **TX: PA2**, **RX: PA3** | 通用串行通信 |
| **I2C1** | **SCL: PB6**, **SDA: PB7** | 用于连接传感器、EEPROM等 |
| **SPI1** | **NSS: PA4**, **SCK: PA5**, **MISO: PA6**, **MOSI: PA7** | 高速传感器或显示屏接口 |
| **ADC1/2** | **PA0 至 PA7** | 12位精度，支持多达10路外部通道 [cite: 375] |
| **USB** | **D-: PA11**, **D+: PA12** | 全速 USB 2.0 接口 [cite: 293] |
| **CAN** | **RX: PB8**, **TX: PB9** | 工业总线通信 [cite: 292] |

### 4. 关键使用注意事项
* **5V 耐压特性**: 大部分 I/O 引脚支持 5V 电平输入（5V-tolerant），但在模拟输入模式或作为某些特定外设时需查阅规格书确认 [cite: 301]。
* **电流驱动能力**: 
    * 通用 GPIO 可吸收或输出最高 **±8mA** 电流 [cite: 90]。
    * **PC13, PC14, PC15** 的驱动能力较弱（仅 **±3mA**），且作为输出使用时速度限制在 2MHz 以下，主要用于低速控制或 RTC 信号 [cite: 90, 91]。
* **特殊引脚 PC13**: 板载蓝色 LED 通常连接在 PC13。该引脚同时兼任侵入检测（TAMPER）和 RTC 校准输出功能 [cite: 41]。
* **PA0 (WKUP)**: 具有唤醒功能，可用于将 MCU 从待机模式唤醒 [cite: 382]。
